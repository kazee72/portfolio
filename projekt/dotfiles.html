dotfiles

Category: tools — lua, config

Overview

A from-scratch macOS desktop setup ("rice") built around a tiling workflow: AeroSpace for window tiling, SketchyBar for a custom status bar, boringNotch for the notch, and kitty as the terminal — all themed from a single centralized colour palette.

Features

  • AeroSpace tiling window manager configuration
  • custom SketchyBar status bar (workspace pills, focused-window name)
  • boringNotch + kitty terminal integration
  • centralized colors.sh palette for one-file theming
