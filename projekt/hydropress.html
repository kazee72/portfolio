hydropress

Category: tools — rust

Overview

hydropress is a file-compression CLI built in Rust around Huffman coding. It packs files into a custom .h2 binary format and unpacks them losslessly, with a modular design separating the coder from file I/O.

Features

  • Huffman encoding/decoding to a custom .h2 binary format
  • modular architecture (huffman.rs, file_io.rs)
  • handles edge cases: empty files and single-unique-byte inputs
  • 11 passing tests (logic + CLI)
  • published to crates.io as v0.1.1
