acrux

Category: systems — rust

Status: early development

Overview

acrux is a custom programming language with a tree-walking interpreter written in Rust. The pipeline runs Lexer → Parser → AST → Evaluator, using Pratt parsing for expression precedence and boxed nodes for the recursive AST.

Features

  • hand-written lexer and parser
  • Pratt parsing for expression precedence
  • Box<T> for recursive AST node types
  • tree-walking evaluator
