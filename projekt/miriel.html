miriel

Category: bots — node.js, discord.js

Overview

Miriel is a Discord wiki bot for Elden Ring. On command it surfaces items, NPCs, and characters — pulling data from a community-made Elden Ring API and images from the Fextralife wiki, with a direct link back to each entry.

Features

  • item, NPC, and character lookups via Discord commands
  • data from a community-made Elden Ring API
  • images pulled from the Fextralife wiki
  • direct links back to each item/NPC page
