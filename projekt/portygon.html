Portygon

Security
Rust tokio clap serde

Overview

Portygon is an asynchronous TCP port scanner written in Rust. It scans port ranges with bounded concurrency on the tokio runtime, grabs service banners, supports stealth timing, and can emit results as JSON for tooling pipelines.

Features

  • tokio async scanning with semaphore-bounded concurrency (limit 100, tuned down from 200 for reliability)
  • clap-based CLI with a progress bar (indicatif)
  • banner grabbing: passive, plus an HTTP GET probe for known web ports
  • --stealth flag with randomized sequential delays
  • --json output via serde; colored, sorted terminal output
  • 14 passing tests (targets: 127.0.0.1 and scanme.nmap.org)

Roadmap

  • hostname resolution
  • scan summary with elapsed time
  • --timeout flag
  • adaptive concurrency
