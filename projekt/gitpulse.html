gitpulse

Category: tools — bash

Overview

GitPulse is a Bash CLI that scans local Git repositories and reports their health — uncommitted changes, stale branches, sync status — with cron integration for scheduled checks.

Features

  • scans multiple Git repos for health status
  • modular Bash architecture
  • cron integration for scheduled runs
  • 20-test suite
  • Word / Markdown documentation
